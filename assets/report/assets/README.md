# Progress report assets

Media is organized by report week. Each week folder contains:

- The photos, GIFs, and videos used by that week in `Progress_Report.html`, don't crop them.
- A markdown file with the same week name, such as `week_8.md`, that mirrors the report content for easier future updates.

## Folder convention

- `week_1_2/`
- `week_3/`
- `week_4/`
- `week_5_6/`
- `week_7/`
- `week_8/`

For a new week, create a folder such as `week_9/`, put the media files there, add `week_9.md`, and update the matching media paths in `../Progress_Report.html`.
